# dotfiles

These are my dotfiles highly inspired by various sources.

## Requirements

- zsh, macOS, ...

## Installation
- Clone the repository as `~/.dotfiles`
- Run `make all` and restart your system
- Create `~/.local.gitconfig` and `~/.local.zsh` for additional local settings
